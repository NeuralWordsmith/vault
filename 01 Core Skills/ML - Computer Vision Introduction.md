### 1. Definition of Computer Vision

Computer vision is a field of artificial intelligence that enables machines to **interpret, analyze, and understand visual information** from images and videos, similar to how humans perceive the visual world.

---

### 2. Visual Data as Input

Computer vision systems work with visual data such as:

* Images
* Video frames
* Pixel-based representations

These inputs are converted into numerical form so that models can process them.

---

### 3. Core Computer Vision Tasks

Common tasks in computer vision include:

#### a. Image Classification

* Assigning a label to an entire image
* Example: Identifying whether an image contains a cat or a dog

#### b. Object Detection

* Identifying and locating objects within an image
* Output includes object labels and bounding boxes

#### c. Image Segmentation

* Assigning a label to each pixel in an image
* Used for detailed scene understanding

---

### 4. Role of Deep Learning in Computer Vision

Modern computer vision relies heavily on deep learning, especially **convolutional neural networks (CNNs)**, which:

* Automatically learn spatial features
* Capture edges, shapes, and textures
* Perform well on high-dimensional image data

---

### 5. Applications of Computer Vision

Computer vision is used across many domains, including:

* Facial recognition
* Medical imaging
* Autonomous driving
* Surveillance systems
* Quality inspection in manufacturing

---

### 6. Challenges in Computer Vision

Key challenges include:

* Variations in lighting, angle, and scale
* Occlusion and background noise
* High computational requirements

Robust models must generalize across these variations.

---

### 7. Key Takeaways

* Computer vision enables machines to understand visual data
* Images and videos are processed as numerical representations
* Deep learning drives modern vision systems
* Applications span multiple real-world industries
