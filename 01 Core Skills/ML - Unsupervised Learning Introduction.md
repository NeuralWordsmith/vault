### 1. Definition of Unsupervised Learning

Unsupervised learning is a type of machine learning where the model is trained on **unlabeled data**. The system is not given correct answers and must discover structure and patterns on its own.

---

### 2. Nature of Unlabeled Data

In unsupervised learning:

* Data contains no predefined outputs
* The model explores similarities and differences between data points
* Learning is driven by inherent data structure rather than explicit guidance

This makes unsupervised learning more exploratory in nature.

---

### 3. Common Unsupervised Learning Tasks

Unsupervised learning is commonly used for:

#### a. Clustering

* Groups similar data points together
* Examples:
  * Customer segmentation
  * Document grouping
  * Image similarity analysis

#### b. Dimensionality Reduction

* Reduces the number of features while preserving important information
* Helps with visualization and efficiency
* Examples include feature compression and noise reduction

---

### 4. Pattern Discovery

Unsupervised learning helps uncover hidden patterns that may not be obvious to humans. These patterns can reveal:

* Natural groupings
* Anomalies or outliers
* Underlying data distributions

---

### 5. Use Cases

Typical applications include:

* Market and customer analysis
* Recommendation systems (as a supporting technique)
* Data exploration and preprocessing for supervised learning

---

### 6. Challenges of Unsupervised Learning

* No clear ground truth for evaluation
* Results can be harder to interpret
* Quality depends heavily on data representation

---

### 7. Key Takeaways

* Unsupervised learning works without labeled data
* Focuses on discovering structure and patterns
* Common techniques include clustering and dimensionality reduction
* Often used for exploration and feature learning
