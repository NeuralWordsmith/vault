# Core: Boolean Indexing in NumPy

## Summary

>Boolean indexing is a powerful feature in [[Python - NumPy (Numeric Python)|NumPy]] that allows for the selection of elements from an array based on a set of `True`/`False` values, known as a boolean mask. This mask is typically generated by applying a conditional expression directly to the array, resulting in a concise and performant alternative to traditional loops or [[Python - List Comprehensions|list comprehensions]].

**Why This Matters:** Boolean indexing provides a highly efficient and readable way to filter large datasets, which is fundamental for data cleaning and analysis tasks in scientific computing.

_Analogy:_ _Imagine you're a bouncer at a concert with a list of all invited guests (the NumPy array). Your job is to only let in people who are 21 or older (the condition). Instead of checking each person's ID one-by-one and pointing them to the entrance (a `for` loop), you first go down the entire list and put a checkmark next to the name of every person who is 21 or older. This list of checkmarks is your boolean mask. You then hand this checked list to the door staff, who can instantly see who to let in, admitting the entire qualified group at once._

The bouncer's check is the conditional logic (`array > 21`), the list of guests is the NumPy array, the checkmarks are the boolean mask (`[True, False, True, ...]`), and the group of people let in is the resulting filtered array. **Where it breaks down:** This analogy implies a two-step manual process. In NumPy, the creation of the mask and the filtering are so highly optimized that they feel like a single, instantaneous operation, unlike the bouncer having to first create the list and then act on it.

```
Original Array:  [-10,  -5,   0,   5,  10,  15]
                     │    │    │    │    │    │
Condition:         ( > 0? )   ( > 0? ) ...
                     │    │    │    │    │    │
                     ▼    ▼    ▼    ▼    ▼    ▼
Boolean Mask:    [False,False,False,True,True,True]
                                  │    │    │
                                  └────┼────┘
                                       ▼
Result Array:                  [  5,  10,  15]
```

## Details

In [[Python - NumPy (Numeric Python)|NumPy]], boolean indexing is a specialized technique for filtering data. It leverages vectorization to apply a condition across an entire [[Python - NumPy Array 1|NumPy array]] at once, which generates a new array of boolean values (`True` or `False`). This "mask" array has the same shape as the original, and when used as an index, it efficiently pulls out only the elements from the original array that correspond to a `True` value. This approach is significantly faster and more readable than the equivalent operation on a standard Python list, which would necessitate an explicit `for` loop or a list comprehension.

#### Primary Goal

To select and retrieve elements from a NumPy array that satisfy a specific logical condition in a highly efficient, vectorized manner.

#### Mechanism

- **Step 1: Create the Source Array**
    - Start with a NumPy array containing the data you wish to filter.
- **Step 2: Define a Boolean Condition**
    - Write a logical expression that will be evaluated for each element in the array (e.g., `array > 0`).
- **Step 3: Generate the Boolean Mask**
    - Apply the condition directly to the array. NumPy performs this operation element-wise, returning a new array of the same shape containing only `True` or `False` values.
- **Step 4: Apply the Mask for Indexing**
    - Use the boolean mask array inside the square brackets of the original array (`original_array[mask]`). This returns a new array containing only the elements from the original array where the mask had a `True` value.

##### Code Translation

```python
import numpy as np

# --- Step 1: Create the Source Array ---
# This is our original dataset.
data_array = np.array([-10, -5, 0, 5, 10, 15])

# --- Step 2: Define a Boolean Condition ---
# We want to find all numbers greater than 0.
# The condition is `data_array > 0`.

# --- Step 3: Generate the Boolean Mask ---
# Applying the condition to the array creates a new array of booleans.
# Where the condition is met, the value is True; otherwise, it's False.
positive_mask = data_array > 0
# print(positive_mask) -> [False False False  True  True  True]

# --- Step 4: Apply the Mask for Indexing ---
# Use the mask inside the square brackets to select elements from the original array.
# Only elements at indices where the mask is True will be returned.
positive_numbers = data_array[positive_mask]

print(f"Original Array: {data_array}")
print(f"Boolean Mask (is > 0?): {positive_mask}")
print(f"Filtered Array: {positive_numbers}")
```

#### Key Parameters

- **The Array:** The source NumPy array you want to filter.
- **The Condition:** A logical expression that can be evaluated for each element in the array. This can involve comparison operators (`>`, `<`, `==`) and logical operators (`&` for AND, `|` for OR, `~` for NOT).
    - Example: To find numbers between 0 and 10, the condition would be `(my_array > 0) & (my_array < 10)`.

#### Core Trade-offs

- **Performance vs. Memory**
    - Pro: Boolean indexing is implemented in C and is significantly faster than Python loops for large arrays due to vectorization.
    - Con: It requires creating an intermediate boolean array (the mask) that is the same size as the original array. For extremely large arrays, this can temporarily double the memory usage.
- **Copy vs. View**
    - Boolean indexing always returns a *copy* of the data, not a view. This is a safety feature, as modifying the new array will not accidentally change the original data. However, if you need to modify the original array, you must use a different indexing method or reassign the modified data back.
- **Readability**
    - Pro: The syntax `array[condition]` is highly expressive and easy to read, making the code's intent clear at a glance compared to a multi-line loop or a complex list comprehension.
